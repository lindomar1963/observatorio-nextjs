# Observatório de Segurança Pública do Amazonas — ALEAM

Portal institucional do Observatório de Segurança Pública da Assembleia Legislativa do Estado do Amazonas.

## Stack
- **Framework:** Next.js 14 (App Router)
- **Linguagem:** TypeScript
- **Estilização:** Tailwind CSS
- **Deploy:** Vercel (gru1 — São Paulo)
- **DNS:** observatoriodeseguranca.com

## Padrões
- LGPD — Lei Geral de Proteção de Dados
- eMAG — Modelo de Acessibilidade em Governo Eletrônico
- WCAG 2.2 — Diretrizes de Acessibilidade
- Design System Gov.br

## Setup local
```bash
npm install
npm run dev
```

## Deploy
Conecte ao Vercel e aponte o DNS do Hostinger para os nameservers da Vercel.
