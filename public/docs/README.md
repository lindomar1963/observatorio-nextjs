# Documentos de Transparência

Coloque os PDFs nesta pasta com **exatamente** estes nomes. Assim que o arquivo
existir aqui, o item correspondente na página `/transparencia` vira um link de
download automaticamente (nenhuma alteração de código é necessária).

## Observações

- Os itens "Orçamento" e "Contratos" apontam para o Portal ALEAM (link externo).
- Os itens "Em elaboração" não precisam de arquivo enquanto não concluídos.
- Para adicionar novos documentos: inclua o campo `arquivo` no item em
  `src/app/transparencia/page.tsx` e coloque o PDF aqui com o mesmo nome.
