# Documentos de Transparência

Coloque os PDFs nesta pasta com **exatamente** estes nomes. Assim que o arquivo
existir aqui, o item correspondente na página `/transparencia` vira um link de
download automaticamente (nenhuma alteração de código é necessária).

| Documento | Nome do arquivo esperado |
|-----------|--------------------------|
| Resolução de criação — ALEAM nº 003/2023 | `resolucao-criacao-003-2023.pdf` |
| Relatório de atividades 2024 | `relatorio-atividades-2024.pdf` |
| Protocolo de cooperação — UFAM | `protocolo-cooperacao-ufam.pdf` |
| Protocolo de cooperação — SSP-AM | `protocolo-cooperacao-ssp-am.pdf` |

Os itens "Orçamento" e "Contratos" apontam para o Portal ALEAM (link externo) e
não precisam de arquivo. Os itens "Em elaboração" passam a ter arquivo quando
concluídos — basta adicioná-los aqui e marcar `status: 'Disponível'` com o
campo `arquivo` em `src/app/transparencia/page.tsx`.
